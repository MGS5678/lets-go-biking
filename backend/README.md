Il faut utiliser les fichiers .bat pour lancer les differents projets.

Lancer les serveurs (routeur et proxy)
run_servers.bat

Lancer le light client
run_light_client.bat

Lancer le heavy client
run_heavy_client.bat

Pour le heavy client, il s'agit simplement d'un programme qui demande à l'utilisateur le point de départ et d'arrivée, puis le chemin à prendre est affiché (tout dans le terminal).
